CNC Pen Plotter

Arduino · GRBL · SolidWorks

A fully custom-built 2-axis CNC pen plotter designed and fabricated from scratch. The frame was modeled in SolidWorks and 3D printed in PLA. Motion is driven by NEMA 17 stepper motors controlled via an Arduino Nano running GRBL-Servo firmware, with A4988 drivers and a GT2 belt system calibrated to 81 steps/mm. The workflow converts SVG artwork into G-code using svg2gcode, then streams it to the machine via LaserGRBL. The plotter has successfully reproduced detailed line art including portrait work and hand-lettered text.

Lithophane Lamp

3D Print · Electronics · FastLED

A handmade rechargeable lithophane lamp built as a gift. The enclosure was designed in SolidWorks and 3D printed in PLA, housing an 18650 battery charged via a TP4056 module and boosted to operating voltage by an XL6009E1. A WS2812 LED ring driven by an Arduino Nano and the FastLED library provides 8 selectable color modes with potentiometer-controlled brightness and a long-press random color function. Significant troubleshooting went into resolving LED failures caused by voltage spikes from the boost converter, resolved by adding filtering capacitors and tuning the output voltage.
